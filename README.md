# nekoy's lily58 layout

## Key features

- Qwerty, Colemak
- Dedicated gaming layer with arrow arrow cluster
- Navigational layer with vim motions
- Numberpad

## Build instructions

- `cd /path/to/qmk_firmware`
- Ensure latest libraries are loaded `make git-submodule`
- you have to pass either `-e SIDE=RIGHT` or `-e SIDE=LEFT` depending on the side you want to flash
